## 集成扩展
  1. [Chrome商店](https://chrome.google.com/webstore/category/extensions?hl=zh-CN)添加扩展
  2. [查看扩展ID](chrome://extensions)
  3. 打开扩展安装目录
    - MAC: /Users/{mac_user}/Library/Application Support/Google/Chrome/Default/Extensions
